# Tic Tac Toe Game

A simple web-based Tic Tac Toe game implemented in HTML, CSS, and JavaScript.

## Features
- Classic 3x3 game board
- Player X goes first, followed by Player O
- Win detection for all possible winning combinations
- Draw detection when the board is full
- Visual highlighting of winning moves
- Reset button to start a new game

## How to Play
1. Player X always goes first
2. Click on any empty square to place your mark
3. Players alternate turns until someone gets three in a row (horizontally, vertically, or diagonally)
4. The game will announce the winner or a draw
5. Click "Reset Game" to start a new match

## Installation
Simply open `tic-tac-toe.html` in any web browser to play the game.
